Place your APK file here with the name: kava_app.apk

The download link on the website will point to this file.
When you build your Flutter APK, place the compiled file (usually found in build/app/outputs/flutter-apk/app-release.apk) 
and rename it to "kava_app.apk" in this directory.

Example:
- Source: build/app/outputs/flutter-apk/app-release.apk
- Destination: server/public/downloads/kava_app.apk
